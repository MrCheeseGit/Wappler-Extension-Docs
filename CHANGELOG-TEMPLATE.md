# Changelog template (single current version)

Use one `##` section matching `package.json` `version`. On each release, replace the file with a new single section (do not accumulate history in-repo after a history reset).

```markdown
# Changelog

## {VERSION}

Current release of **{EXTENSION_NAME}**. See [README](README.md) for features, installation, and usage.

Licensed under [Mr Cheese Extension License v1.0](https://www.mrcheese.co.uk/extension-license). Earlier MIT releases, if any, remain under MIT at their historical version tags.
```
