# Mr Cheese Wappler extension docs

Shared compatibility notes for [Mr Cheese Wappler extensions](https://www.mrcheese.co.uk/extensions): how they work together in one project, common stacks, and troubleshooting.

Each extension has its own GitHub repo with install steps, examples, and `wappler-install.json`.

**Install extensions:** [mrcheese.co.uk/extensions/install](https://www.mrcheese.co.uk/extensions/install)  
**Browse the catalog:** [mrcheese.co.uk/extensions](https://www.mrcheese.co.uk/extensions)

## Linking from an extension README

If you maintain or adopt a Mr Cheese extension, you can add a short **Compatibility** section:

```markdown
## Compatibility

Standalone extension. For shared patterns (Redirect-IT step order, PuSH-IT, optional pairs), see
[Mr Cheese extension docs](https://github.com/MrCheeseGit/Wappler-Extension-Docs/blob/main/extension-compatibility.md).
```

Adjust the first sentence if the extension is not standalone (for example Redirect-IT or PuSH-IT).
